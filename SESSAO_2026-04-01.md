# Sessão de Desenvolvimento — 2026-04-01

## O que foi construído

Projeto SaaS completo de geração de vídeo com motion transfer via IA.

**Diretório:** `C:\Users\Pichau\Downloads\Claude\motion-transfer-saas\`

---

## Stack

- Next.js 15 + App Router + TypeScript
- Tailwind CSS + shadcn/ui (componentes manuais)
- Framer Motion (animações)
- React Hook Form + Zod (validação)
- better-auth (autenticação email+senha)
- PostgreSQL + Prisma ORM
- Redis + BullMQ (fila assíncrona)
- Storage local (dev) / S3-compatible (prod)
- AI Providers: Mock, Replicate, ComfyUI

---

## Estrutura do Projeto

```
motion-transfer-saas/
├── apps/

```

## Como Rodar (do zero)

```bash
cd motion-transfer-saas

# 1. Instalar dependências
npm install

# 2. Subir banco e redis
docker-compose up -d

# 3. Criar tabelas e seed
cd packages/database
npm run db:push
npm run db:seed
cd ../..

# 4. Copiar env
# (já existe apps/web/.env.local com valores padrão para dev)

# Terminal 1 — Frontend
cd apps/web
npm run dev
# → http://localhost:3000

# Terminal 2 — Worker
cd apps/worker
npm run dev
```

## AI Providers

Configurar no `.env` via `AI_PROVIDER=`:

| Valor | Descrição |
|---|---|
| `mock` | Simula com delay, copia os arquivos (dev) |
| `replicate` | Stable Video Diffusion via Replicate API |
| `comfyui` | ComfyUI self-hosted |

### Para usar Replicate:
```env
AI_PROVIDER="replicate"
REPLICATE_API_TOKEN="r8_sua_chave_aqui"
```

### Para usar ComfyUI:
```env
AI_PROVIDER="comfyui"
COMFYUI_URL="http://localhost:8188"
```
